**TRIPWAY** is a high-end, "all-in-one" travel planning platform designed to optimize the tourist experience. It features a seamless user experience focused on modern route mapping, real-time discovery of services, and a fluid glassmorphism interface for building a perfect travel itinerary.

---

## 🚀 Features

- **Smart Route Planning:** Interactive map-based dashboard to build custom travel itineraries from scratch with high precision.
- **Intelligent Search:** Integrated auto-complete system (via Nominatim API) to locate cities, specific neighborhoods, or hotels globally.
- **Dynamic POI Discovery:** Real-time exploration of verified tourist attractions, restaurants, and essential services within a 5km radius.
- **Manual Pin Adjustment:** Advanced "Home Base" management allowing users to manually drag and drop markers for perfect localization.
- **Premium UI/UX:** A modern storefront focusing on high-end typography, glassmorphism effects, and fluid responsiveness across all pages.
- **Itinerary Management:** Secure dashboard to manage travel dates, persistent route data, and protected sessions for authenticated users.
